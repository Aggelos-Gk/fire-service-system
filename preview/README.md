# Fire Service System UI Preview

This directory contains curated screenshots that present the Fire Service System user interface and its core operational workflows.

## Included Views

- Dashboard and command overview screens showing incidents, key status indicators, and primary actions.
- Incident workflow screens demonstrating creation, updates, and case management flow.
- Resource and response monitoring screens showing operational status and live coordination context.

## Purpose

The images in this folder provide a quick visual reference for stakeholders, reviewers, and contributors who need to understand the product experience without running the application locally.
